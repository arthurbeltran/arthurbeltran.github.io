Invenciones De La Inventiva Website

About the Project

This website focuses on Spanish language translations of the episodes of the radio program, Engines of Our Ingenuity. The website contents are arranged in episodes and contains a wide variety of teaching materials such as videos, printable content and interactive activities.

My role in the project were to develop the overall design of the website, structure of the contents, development of th interactive activities and the back-end interface for content maintenance. Skills utilized in the project were...

  • Instructional design using Rapid Prototyping method.
  • Project management of the materials in collaboration with the subject-matter experts.
  • Content design using HTML and CSS.
  • Video conversion using Handbrake and other similar tools.
  • Graphic design using Photoshop.
  • Website development using HTML, CSS, Coldfusion and SQL.
  • Interactive content development using Javascript and Jquery.
